# 月经助手 (Menstrual Helper)

一款专注于女性经期健康管理的 UniApp 应用，极简、私密、无广告、纯本地离线优先。

## 功能特性

### 🌸 核心功能

- **快速记录** - 一键开始/结束经期，记录流量、颜色、痛感、症状、情绪、用药
- **日历视图** - 按月展示经期、排卵期、易孕期、安全期
- **智能预测** - 基于历史周期自动预测下次经期、排卵日
- **数据统计** - 平均周期、经期天数、周期稳定性、异常提示
- **隐私保护** - 支持密码锁，纯本地存储

### 📱 页面结构

1. **首页** - 今日状态 + 快速记录 + 近期预测
2. **日历页** - 月视图展示周期状态
3. **记录页** - 添加/编辑经期数据
4. **统计页** - 周期分析图表
5. **我的** - 设置、隐私、备份

### 🎨 设计特点

- 柔和清新 UI，主色淡粉/淡紫
- 卡片布局，圆角设计
- 极简操作，打开即用
- 支持深色模式（跟随系统）

## 技术栈

- **框架**: UniApp (Vue 2)
- **平台**: Android + iOS + H5 + 小程序
- **存储**: 本地 uni.storage
- **预测算法**: 基于历史数据的平均值算法

## 使用说明

### 开发环境

1. 安装 HBuilderX
2. 导入项目
3. 运行到浏览器或模拟器

### 打包发布

#### Android
1. 在 HBuilderX 中选择"发行" -> "原生App-云打包"
2. 配置 Android 包名和签名
3. 生成 APK

#### iOS
1. 在 HBuilderX 中选择"发行" -> "原生App-云打包"
2. 配置 iOS Bundle ID 和证书
3. 生成 IPA

#### H5
```bash
npm run build:h5
```

#### 微信小程序
```bash
npm run build:mp-weixin
```

## 预测算法说明

### 周期预测
- 计算历史周期长度的平均值
- 预测下次经期 = 最近经期开始日 + 平均周期天数
- 置信度评估：基于周期波动标准差

### 排卵日预测
- 排卵日 = 预测经期开始日 - 14天
- 易孕期 = 排卵日前5天到后1天

### 周期稳定性评分
- 标准差 ≤ 1天：100分（非常规律）
- 标准差 ≤ 2天：90分
- 标准差 ≤ 3天：80分
- 标准差 ≤ 5天：60分
- 标准差 ≤ 7天：40分
- 标准差 > 7天：20分
